Plugins, themes and the core update can be disabled from the setting page.

== Description ==

Disable update options for theme, plugins, WordPress core and auto updates. To prevent losing any changes that has been made in code files by disabling theme/plugin future updates.

### Features And Options:
* Disable Themes update option
* Disable Plugins update option
* Disable WordPress Core update option
* Disable Automatic update option

== Keywords ==
disable theme update, disable plugins update, disable wordpress update, auto update

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/disable-updates-for-wordpress` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the "Disable Updates by CV" plugin through the 'Plugins' screen in WordPress
3. Click on "WP Disable updates" on the dashboard.
